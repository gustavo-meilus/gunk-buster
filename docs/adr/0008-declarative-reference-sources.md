# Custom reference sources are declarative

Custom trusted reference sources are described by configuration rather than executable plugins. A definition selects source files, extracts targets with deterministic format-specific selectors, and declares whether targets resolve from the source directory or repository root. JSON and YAML selectors support named properties, array indices, and `*` wildcards over object values or array elements; multiple selectors are allowed, but recursive descent, predicates, and transformations are not. Text extraction requires an explicit regular expression with a named `target` capture. Only string selector results are targets; every non-string match emits a configuration diagnostic without preventing valid sibling matches from being evaluated. A source file glob matching nothing is likewise diagnostic; configured sources are never silently optional. A malformed source or unevaluable selector emits a visible configuration diagnostic in human and JSON output, produces no assertions from the invalid input, and never falls back to raw text scanning; the read-only scan continues under the existing advisory exit-code policy. A successfully extracted target emits a reference assertion only when it resolves in the current repository inventory; otherwise it emits a broken-reference finding attributed to the source and target. This gives custom manifests and topology files an extensible reference contract without allowing configuration to execute arbitrary code or undermine Gunk Buster's deterministic, local-only behavior.

Definitions live under `references.sources` in `gunk.config.json`. Each has a stable diagnostic `name`, one or more source `files`, an explicit `format`, one or more `selectors`, and `resolveFrom` set to `source-directory` or `repository-root`.

The graph retains every distinct source, selector, and location that asserts the same target, while exact duplicate assertions are collapsed. GHOST needs only one valid inbound assertion, but reports and diagnostics preserve every route that proves liveness.
